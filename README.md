
  

# React Quantity Picker

  

## Installation

  ### NPM
```bash

$ npm install --save react-qty-picker

```
### Yarn
```bash

$ yarn add react-qty-picker

```

  

## Usage

  

### Import

  

```bash

import { QuantityPicker } from 'react-qty-picker';

```

### Use

  

```bash

<QuantityPicker />

```
